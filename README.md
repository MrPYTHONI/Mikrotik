## ⚡ Mikrotik ⚡🕷️🖤

في ظلال الظلام، وبين أسطر الأكواد المخفية، تتربص أداة Mikrotik Bruteforce. هي الوحش الذي يفتش بين كلمات السر، يبحث عن ثغرات الحماية كما يبحث الصياد عن فريسته. هذه الأداة ليست مجرد سكربت عادي، بل هي سلاح إلكتروني ينقلك لعالم من الهجوم الصامت، حيث تهمس الشاشات بأصوات الأوامر، وتضيء أضواء الكونسول باللون الأحمر القاني كأنها تحذير لا يُستهان به.

مع كل محاولة تخمين، تشعر وكأنك تخترق جدران مجهولة، تجوب ممرات مجهولة في قلب أنظمة Mikrotik، تاركًا خلفك أثرًا لا يمحى... هل أنت مستعد لتشغيل الوحش؟

options:
 
  ```-h, --help```    show this help message and exit
  
  ```-u URL, --url URL ```    Base URL or Gateway URL of the hotspot login page
  
  ```-f FILE, --file FILE ``` File with cards to try (one per line)
  
  ```-t THREADS, --threads``` THREADS
                        Number of threads for parallel attempts

 ``` --delay DELAY ```        Delay between attempts in seconds
 
 ``` --verbose   ```          Verbose output



 
⛓️ شرح استخدام سكربت تخمين كروت ميكروتيك ⛓️ 🛠️ 

1- قبل تشغيل السكربت، تأكد من وجود التالي: 

🔹 المكتبات المطلوبة:

requests bs4 ← (BeautifulSoup) netifaces ← (اختياري للكشف التلقائي عن IP البوابة) 


✅ إذا كنت في Termux، نفّذ هذا:

```bash
pkg update && pkg install python -y pip install requests bs4 netifaces 
```


 2- أوامر التشغيل: 

🧾 الصيغة الأساسية للتشغيل:

python3 script.py -u <base_url> -w <cards.txt> [خيارات إضافية] 

✅ الشرح الكامل للأوامر:

الخيار المعنى -u أو --url رابط تسجيل الدخول أو IP البوابة (مثلاً: http://192.168.88.1/)

 -w أو ```--wordlist``` ملف يحتوي على كروت التخمين (واحد في كل سطر) 
 
 -t أو ```--threads``` عدد الثريدات (افتراضي: 10) — لتسريع عملية التخمين 
 
 -d أو ```--delay``` وقت تأخير بين كل محاولة (بالثواني، افتراضي: 0.5)
 
 -v أو ```--verbose``` طباعة تفصيلية لكل المحاولات الناجحة والفاشلة 🧪 
   
 3- مثال عملي للتشغيل: 🔸
 
  1. رابط البوابة: 

افترض أن صفحة تسجيل الدخول هي:

http://192.168.88.1/ 


 2. إنشاء ملف الكروت: 

أنشئ الملف بالأمر:

```bash
nano cards.txt
```

📥 أضف بداخله كروت بالشكل:

123456 

admin guest freeuser 

 3. تشغيل السكربت: 
 
 python3 script.py -u http://192.168.88.1/ -w cards.txt -t 10 -d 1 -v 
 
 4. إذا تم العثور على كرت صحيح: 

سيظهر:

[SUCCESS] Working card found: 123456 

وسيتم حفظ الكرت الناجح في الملف:

```working.txt ```
💡 ملاحظات مهمة: 

🔒 لا تستخدم السكربت على شبكات لا تملك إذنًا بها.

🕒 لتفادي الحظر من صفحة الميكروتيك، استخدم -d 1 أو أعلى لزيادة التأخير.

🧠 إذا لم تدخل الرابط بشكل صحيح، سيحاول السكربت اكتشاف الصفحة تلقائيًا.

🧾 مثال إضافي بتأخير أكبر:

 python3 script.py -u http://192.168.1.1/ -w cards.txt -t 5 -d 2 -v

⚙️ 1- وظيفة السكربت: 

سكربت بلغة Python يقوم بـ:

(3 كروت حقيقية تعطيها له). 🧠 توليد عدد كبير من الكروت المشابهة بناءً على هذا النمط. 💾 حفظ الكروت في ملف للاستخدام لاحقًا في التخمين. 💡


 2- خطوات الاستخدام: 🚀 قبل التشغيل: 

تأكد من أنك تملك:

Python 3.6 أو أحدث. مكتبة colorama (سيتم تثبيتها تلقائيًا عند أول تشغيل). 🧠 

أمر التشغيل:

```bash

```


 3- خطوات التنفيذ داخل السكربت: 🎬 عند التشغيل: 

يعرض بانر مرعب وواجهة هاكر أنيقة.

✍️ يطلب منك: 

```[*] Enter 3 sample cards (same length): ```

📌 أدخل 3 كروت (مثلًا):

```Card 1 ➤ 123456 ```

```Card 2 ➤ 128999 ```

```Card 3 ➤ 120321 ```

📌 يجب أن تكون الكروت بنفس الطول (وإلا يظهر لك: 

```❌ Cards must be of the same length!)```

🧮 ثم يسألك: 

```How many cards to generate? ➤ ```

🧾 أدخل الرقم الذي تريده مثل: ```1000```

![Screenshot_20250604-173751_Termux](https://github.com/user-attachments/assets/c13c1461-6328-4978-9f09-eeab49691e70)

🧠 4- طريقة التحليل: 

يقوم السكربت بتحليل كل حرف في الكروت:

النوع الشرح fixed حرف ثابت بكل الكروت

 ```rand_digit``` رقم يختلف من كرت لآخر 
 
``` rand_lower``` حرف صغير متغير 
 
 ```rand_upper``` حرف كبير متغير 
 
 ```rand_mix``` أي حرف أو رقم متغير 

✅ ثم يولّد عدد كبير من الكروت المشابهة تمامًا من حيث البنية.
